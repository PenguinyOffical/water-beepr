## Important
This project requires the Climate Action Land Kit and a BBC Microbit V1 or V2
## Use as Extension

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/penguinyoffical/water-beepr** and import

## Edit this project

To edit this repository in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/penguinyoffical/water-beepr** and click import
